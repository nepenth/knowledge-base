
## Main Concepts
The main concept behind JSON Crack is the transformation of data formats into graph structures that can be interactively explored. This is achieved through a combination of format conversion capabilities, advanced tooling, and client-side processing.

### Format Conversion Capabilities
JSON Crack supports the conversion of various data formats, including:

* JSON (JavaScript Object Notation)
* YAML (YAML Ain't Markup Language)
* CSV (Comma Separated Values)
* XML (Extensible Markup Language)
* TOML (Tom's Obvious, Minimal Language)

The tool provides built-in validation and beautification for these formats, ensuring that the data is accurate and easy to read.

### Advanced Tooling
JSON Crack includes advanced tooling capabilities, such as:

* **TypeScript/Golang code generation**: Generate code in TypeScript or Golang based on the transformed graph structure.
* **JWT decoder**: Decode JSON Web Tokens (JWT) for secure authentication and authorization.
* **JSON Schema generation**: Generate JSON Schemas from the transformed graph structure for validation and documentation purposes.
* **jq query support**: Use jq queries to filter and transform the data in the graph structure.

### Client-Side Processing
One of the key features of JSON Crack is its client-side processing capability. This means that all data processing and transformation occur on the client-side, ensuring zero server storage and enhanced data privacy.
